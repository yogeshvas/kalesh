## Tech Stack

**Client:** React Native Expo, NativeWind
**Server:** Appwrite, Cloudinary

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
